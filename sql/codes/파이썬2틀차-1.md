## ✅ 1. replace()
- 문자열의 특정 부분을 다른 문자로 바꾸는 함수

```python

print(s)

복사

✅ 요약 정리표
문법	역할
replace()	문자열 바꾸기
str()	숫자 → 문자열 (자릿수 분리 가능)
int()	문자열 → 숫자 (계산 가능)
while	조건 반복
pass	코드 없이 넘어가기
continue	다음 반복으로 건너뛰기
break	반복 중단
f-string	문자열 안에 변수 넣기
[]	리스트/문자열 인덱싱
